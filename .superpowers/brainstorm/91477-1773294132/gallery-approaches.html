Gallery Landing Page Directions

Each folder gets its own detail page. The question here is how the main /gallery page should introduce the archive.

A. Featured + Directory
Featured trip hero

Featured + Directory

One highlighted trip sets the tone, while the rest of the folders stay scannable as clean cover cards below.

Pros

  • Strong first impression
  • Fast initial scan
  • Still exposes the full archive

Cons

  • Needs a rule for which trip is featured
B. Equal Directory

Equal Directory

Every folder is treated the same: a calm grid of cover-image cards with no featured destination.

Pros

  • Simplest information architecture
  • No editorial choice needed

Cons

  • Less personality
  • Weaker visual hierarchy
C. Expandable Sections

Expandable Sections

The landing page remains the archive, but folders stay collapsed until a visitor opens one inline.

Pros

  • Keeps everything on one route
  • Less jarring than a giant full-page wall

Cons

  • Still mixes landing and browsing roles
  • Heavier interaction state

Recommendation

Approach A is the strongest fit for this site: it gives /gallery a deliberate editorial feel, reduces visual overload, and still leaves room for a clean folder detail page.
